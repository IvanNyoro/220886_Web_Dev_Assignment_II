Definition

The World Wide Web (WWW) is a system of interconnected documents and other resources, linked by hyperlinks and URLs. It is accessed via the Internet.

The WWW was invented by Tim Berners-Lee in 1989 while working at CERN. It revolutionized the way information is shared and accessed globally.

Key components of the WWW include:


  • Web Browsers: Software applications (like Chrome, Firefox, Safari) that allow users to access and navigate the web.

  • Web Servers: Computers that store and serve web pages to users upon request.

  • HTML (Hypertext Markup Language): The standard language used to create and structure web pages.

  • HTTP (Hypertext Transfer Protocol): The protocol used for transmitting web pages over the Internet.
