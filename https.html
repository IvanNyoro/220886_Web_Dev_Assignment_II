Definition

HTTPS (Hypertext Transfer Protocol Secure) is an extension of HTTP that uses encryption (via SSL - Secure Socket Layer/TLS - Transport Layer Security) to secure communication between a user's browser and a web server, ensuring data confidentiality and integrity.

HTTPS is essential for protecting sensitive information (like passwords, credit card details) from being intercepted by attackers during transmission over the internet.

Websites that use HTTPS are indicated by a padlock icon in the browser's address bar, and their URLs start with "https://" instead of "http://".
